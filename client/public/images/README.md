# Movie Images

This folder contains movie posters and UI images for the BookMyShow application.

## Movie Posters
- Add movie poster images here (recommended size: 300x450px)
- Use naming convention: `movie-{movie-name}.jpg`

## UI Images
- Logo, icons, and other UI elements
- Recommended formats: SVG for icons, WebP for photos

## Usage
Images can be accessed via:
```javascript
// In React components
<img src="/images/movie-avengers.jpg" alt="Movie Poster" />
```
